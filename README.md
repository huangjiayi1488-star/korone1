## 我最喜欢的一段代码
```python
# 任务一numpy矩阵运算核心代码
import numpy as np
# 定义两个二维矩阵
a = np.array([[1, 2], [3, 4]])
b = np.array([[5, 6], [7, 8]])
# 矩阵加法运算
matrix_add = a + b
# 矩阵乘法运算
matrix_mul = a @ b
# 输出运算结果
print("二维矩阵加法结果：")
print(matrix_add)
print("二维矩阵乘法结果：")
print(matrix_mul)
```

执行 a @ b 可实现矩阵的矩阵乘法运算，执行 a + b 可实现矩阵的逐元素加法运算，在 Jupyter Notebook 中直接运行上述代码块即可输出对应运算结果。
